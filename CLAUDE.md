## Project overview

**Bayesian Adversarial Privacy (BAP)** is a research project introducing a Bayesian decision-theoretic framework for data privacy. The paper is submitted to **Bayesian Analysis**.

Three agents interact:
- **Alice** (mechanism designer): selects a release mechanism $q(d\eta | x)$ balancing utility and privacy
- **Bob** (statistician): infers parameter $\theta$ from the release $\eta$
- **Eve** (adversary): tries to learn about the private data $x$ from $\eta$

Alice minimises an integrated (ex ante) risk: $R_A(\pi, q) = R_B(\pi, q) - \lambda R_E(\pi, q)$.

## Tech stack

- **Python 3.10.13** (enforced via `.python-version` / pyenv)
- Virtual environment in `.venv/` (do NOT use `BAP/`, which is a stale venv)
- Key dependencies: `jax`, `jaxlib`, `numpy`, `matplotlib`, `PuLP` (LP solver), `tqdm`
- Notebooks are Jupyter `.ipynb` files

## Working with the code

```bash
# Setup
pyenv local 3.10.13
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Run notebooks
jupyter notebook notebooks/
```

- Notebooks change directory to the repo root on startup, so figures/tables are saved relative to the repo root.
- Notebook 1 (coin toss) uses exact arithmetic + PuLP for LP optimisation.
- Notebook 2 (Gaussian) uses JAX for numerical integration of integrated risks across 6 release mechanisms (full, null, noisy full, noisy mean, noisy median, one-bit).

## Working with the paper

- Main entry point: `paper/draft_v5_BA_template.tex`
- Sections are in `paper/sections/` and included via `\input{}`
- Author annotation macros: `\ant{}`, `\cam{}`, `\tim{}`, `\xian{}` for inline comments
- The paper uses `imsart` class with `ba` option (Bayesian Analysis journal)
- Supplementary material is in `paper/supplementary_gaussian.tex`

## Conventions

- Figures are PDF, generated by notebooks and saved to `figures/paper/` and `figures/supplementary/`
- Tables are `.tex` fragments generated by notebooks and saved to `tables/`
- The parameter `lambda` ($\lambda$) controls the privacy-utility trade-off
- The parameter `sigma0` ($\sigma_0$) is the prior standard deviation in the Gaussian example
- File naming: `plot_{sigma|tau}_{paper|supplementary}_{mean|max}[_sigma0_X.X].pdf`
